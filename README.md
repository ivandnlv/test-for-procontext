# Тестовое задание на позицию Frontend Developer (Vue.js) для компании ПроКонтекст

Стэк: 
+ Vue 3 (script setup syntax)
+ Pinia
+ Vue Cli (Vite)
+ Scss
+ Typescript

Для запуска: 
```
npm install 

npm run serve
```
